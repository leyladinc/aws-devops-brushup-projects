# AWS DevOps Brush-Up Projects

This repository contains three beginner-friendly AWS DevOps projects built to practice core AWS services.

## Projects

### Project 1 – S3 Static Website
- Created an S3 bucket
- Configured Static Website Hosting
- Applied Bucket Policy
- Uploaded HTML files
- Tested the public website

### Project 2 – EC2 Web Server
- Launched an EC2 instance
- Configured Security Groups
- Installed Apache using User Data
- Verified the web server

### Project 3 – S3 Upload Trigger with Lambda
- Created an S3 bucket
- Created an IAM Role
- Deployed an AWS Lambda function
- Configured S3 Event Notifications
- Verified execution using Amazon CloudWatch Logs

## AWS Services Used

- Amazon S3
- Amazon EC2
- AWS Lambda
- IAM
- Amazon CloudWatch
- AWS CLI
